## Pain / Problem
Operators and trading pods live in disparate tabs—RSS readers, SEC inboxes, Telegram snippets. Pricing changes slip through, filings languish until compliance forwards them, and Monday standups start with guesswork.

## Solution Snapshot
Signal Relay Hub watches the exact feeds you care about, enriches each event with heuristics from SignalFoundry + Flash SEC Arb, and relays a single impact-ranked digest plus JSON/API delivery hooks.

## Feature Columns
1. **Adaptive Feed Collector**  
   Pulls curated company blogs, changelog RSS, and filings lists hourly via OpenClaw cron. YAML-only configuration means adding a new watchlist takes 60 seconds.
2. **Impact Normalizer**  
   Tags every event (pricing, product, funding, regulatory, talent, partnership, security, marketing) and scores urgency 1–5 so GTM leads know what to escalate.
3. **Digest + Delivery**  
   Auto-builds Markdown briefings, Telegram drops, and email blasts. Phase 2 dashboard unlocks live filtering + JSON API for internal tools.
